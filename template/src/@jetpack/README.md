### Issues

Apply labels appropriately for each issues when created:

- `epic`
- `story`
- `bug`
- `feature`
- `feature-request`
- `enhancement`
- `documentation`
- `question`
- `help wanted`

Labels for maintainers:

- wontfix
- invalid
- good first issue
- duplicate
- on-hold

Priorities:

- priority:high
- priority:medium
- priority:low
